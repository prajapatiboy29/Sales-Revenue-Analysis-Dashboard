Sales-Revenue-Analysis-Dashboard
An interactive Sales & Revenue Analysis Dashboard built using Microsoft Excel. The dashboard provides insights into sales performance, revenue trends, top-performing products, and regional sales using Pivot Tables, Pivot Charts, KPIs, and Slicers.

📌 Project Overview
This project is an interactive Sales & Revenue Analysis Dashboard created using Microsoft Excel. It helps analyze sales performance, revenue trends, product performance, and regional sales through visual dashboards and interactive filters.

🚀 Features
Import and analyze sales data
Sales calculation using Excel formulas
Pivot Tables for data summarization
Interactive Pivot Charts
KPI Tracking (Total Sales)
Revenue Trend Analysis
Top Performing Products
Category-wise Sales Analysis
Region-wise Sales Analysis
Interactive Slicers for filtering data
